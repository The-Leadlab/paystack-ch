## 🎯 Goal
Edit any document and have dashboard numbers update automatically.

## 🎨 Visual Indicators

### Editable Fields
- **Dark background** with border = editable
- **Gold border** when focused = you're editing
- **White text** = easy to read

### Buttons
- **Gold button** = save changes
- **+ Add button** = add new line item
- **🗑️ button** = delete line item
- **✓ button** = mark as verified

### Status
- **Pulsing animation** = save button ready
- **Green checkmark** = document completed
- **Success message** = changes saved

## 💡 Pro Tips

1. **Use Tab key** to move between fields quickly
2. **Use search bar** to find items in long lists
3. **Add multiple items** before saving (saves time)
4. **Verify dashboard** after saving to confirm changes
5. **Edit as many times** as needed - no limit

## 🚨 Important Notes

- ✅ Changes are **permanent** (old data is replaced)
- ✅ Dashboard updates **automatically** (no refresh needed)
- ✅ All fields are **editable** (click anywhere)
- ✅ Save button **must be clicked** (changes won't save otherwise)
- ✅ Success message **confirms** save (wait for it)

## 📱 Mobile Users

Same steps work on mobile:
1. Tap document row
2. Tap any field to edit
3. Tap save button at bottom
4. Wait for success message

## ❓ Troubleshooting

**Q: I clicked a field but can't type**
- A: Make sure you clicked directly on the input (not the label)
- A: Try clicking again - field should get gold border

**Q: I edited but dashboard didn't update**
- A: Did you click the save button?
- A: Did you see the success message?
- A: Try refreshing the page

**Q: Save button is grayed out**
- A: Total amount cannot be zero
- A: Enter a valid amount first

**Q: Changes disappeared**
- A: You must click save button before closing
- A: Edits are not saved automatically
